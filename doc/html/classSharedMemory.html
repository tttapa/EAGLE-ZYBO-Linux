Detailed Description

template<class T>
class SharedMemory< T >

A class that manages a mapping to an address in shared memory. It maps the physical address to an address in virtual memory, and manages this pointer, unmapping it when the object goes out of scope.

Template Parameters
TThe type of the shared object to manage.
