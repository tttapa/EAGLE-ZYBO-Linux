18 /** The phase is a data element that expresses what Ketje is doing
19  * - virgin: the only operation supported is initialization, loading the key and nonce. This will switch
20  * the phase to feedingAssociatedData
21  * - feedingAssociatedData: Ketje is ready for feeding associated data, has started feeding associated data
22  * or has finished feeding associated data. It allows feeding some more associated data in which case the phase does not
23  * change. One can also start wrapping plaintext, that sets the phase to wrapping. Finally, one can
24  * start unwrapping ciphertext, that sets the phase to unwrapping.
25  * - wrapping: Ketje is ready for wrapping some more plaintext or for delivering the tag.
26  * Wrapping more plaintext does not modify the phase, asking for the tag sets the phase to feedingAssociatedData.
27  * - unwrapping: Ketje is ready for unwrapping some more ciphertext or for delivering the tag.
28  * Unwrapping more ciphertext does not modify the phase, asking for the tag sets the phase to feedingAssociatedData.
29  */
30 enum Phase {
31  Ketje_Phase_Virgin = 0,
32  Ketje_Phase_FeedingAssociatedData = 1,
33  Ketje_Phase_Wrapping = 2,
34  Ketje_Phase_Unwrapping = 4
35 };
